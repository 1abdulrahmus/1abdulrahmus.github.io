MyJikanBot

I love both manga and anime, hence this became the passion to create my best project as of yet, MyJikanBot. This Bot utilizes the JDA wrapper- a less common wrapper for Discord alongside with the JIKAN REST API to send back anime or manga information from the MyAnimeList database. This includes search queries, to random manga/anime and much more. My hardest challenges was being able to parse the data and collect it into an organized format. I accomplished this by using embeds that presented the data in a list of 5 entries.

Winter Run

One of my early projects but still holds a big part in my development as a software enthusiast. It is this game- this 2d arcade game that showed me how the principles of code and logic work alongside each other. Winter Run is a game that shows a character- a ninja named Mustafa that jumps over many obstacles. Along the way he witnesses an enormounce abundance of bouncing stars- and so he sets his goal to collect as many stars. He gets interrupted by an evil snowman near the end of each fireplace. It is up to you to help Mustafa collect all the stars. Play now at this link: https://winter-run.herokuapp.com/
